
# Prodesk IT — Sprint 1 (The Corporate Brand)

Hello! I'm Ujjawal, and I have successfully built and deployed the landing page module for Prodesk IT, completing all deliverables across Phase 1, Phase 2, and Phase 3.

###  Live Demo & Preview
- **Live URL:** (https://prodesk-theta.vercel.app/) 
 

###  Tech Stack
- **HTML5** (Semantic Architecture)
- **Tailwind CSS** (Integrated locally for Phase 3 Stretch Goals)
- **Vanilla JavaScript** (Theme Controller & Responsive UI logic)
- **Font Awesome** (Icons via CDN)

###  Features Implemented
* **Phase 1 (Base MVP):** Fully responsive layout (Navbar, Hero section, CSS Grid Services module, and Footer).
* **Phase 2 (UX Enhancements):** Sticky navigation bar, micro-interactions/hover-lift effects on cards, and a robust Dark/Light mode toggle.
* **Phase 3 (Optimization):** Complete UI migration to Tailwind CSS with performance hygiene, optimized asset delivery, and multi-viewport testing.

### How to Run Locally
1. Clone this repository to your local machine.
2. Open the root directory.
3. Open `index.html` directly in your browser or run it via the **Live Server** extension in VS Code.
